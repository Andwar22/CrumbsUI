Buttons

Kumpulan tombol dengan pilihan semantik warna untuk kebutuhan website. Cukup tambahkan class "cr-btn" untuk menerapkan style kemudian diikuti dengan class warna, misal tambahkan class "cr-primary" untuk menampilkan tombol dengan style primary.

Beberapa pilihan style warna yang bisa digunakan :

Terdapat juga style outline untuk yang tidak suka warna latar belakang tombol yang terlalu mencolok. Cukup tambahkan "-outline" di akhir class warna.

Style outline ini tidak tersedia untuk "cr-link" karena tombol link secara default sudah tanpa border.

Style class diatas ditujukan untuk tag <button>, namun bisa juga dipakai untuk tag <a> ataupun <input> seperti berikut :

href

Ukuran

Suka ukuran tombol yang lebih kecil? atau yang lebih besar? Cukup tambahkan class "cr-sm" atau "cr-lg" untuk merubah ukuran tombol.

tombol ikon

Style tombol ini dibuat agar tetap konsisten saat di sisipkan icon. Cukup masukkan font icon sebelum atau sesudah teks tombol.

Jika ingin tombol dengan icon saja, tambahkan class "cr-icon" untuk membuat tombol ikon dengan ukuran 1:1.
